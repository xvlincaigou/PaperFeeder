   - Check if the method scales to complex, multi-step scenarios

3. For Data Synthesis papers:
   - Evaluate the quality and diversity of generated data
   - Check if the method improves with iterative refinement

4. For Benchmark papers:
   - Evaluate whether the benchmark measures true capability vs memorization
   - Check for contamination issues and robust evaluation protocols
   - Assess practical utility for model development
