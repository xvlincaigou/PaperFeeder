# User Profile Presets

These are optional starting points for different research directions.

Normal usage:
- Keep editing the active files in `user/`
- Or point `config.yaml` paths at one of these preset folders

Each preset folder mirrors the active user files:
- `research_interests.txt`
- `keywords.txt`
- `exclude_keywords.txt`
- `arxiv_categories.txt`

Preset folders:
- `profiles/frontier-ai-lab`
- `profiles/interpretability-alignment`
- `profiles/coding-agents-reasoning`
- `profiles/multimodal-generative`
